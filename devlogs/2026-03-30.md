### F-CS-A — Change Password for C/S users
- Backend: PATCH /api/auth/change-password in auth_routes.py
  - JWT required, clinician/secretary roles only (403 for others)
  - Verifies current password, enforces 8-char minimum
  - Returns 401 with specific message if current password wrong
  - Transaction boundary with rollback on failure
- Frontend: ChangePassword.jsx at /clinician-dashboard/change-password
  - Three fields: current, new, confirm; show/hide toggles
  - Client-side validation: match + min 8 chars
  - 401 maps to inline error on current password field
  - Success banner auto-dismisses 5s, form resets
- StaffLayout.jsx: "Change Password" added to sidebar nav
- App.jsx: route wired
