To test backend and database migration setup(docker engine is open):
1. Build the backend image: 
docker build .\a-mmc_backend\ -t krispybata/a-mmc-backend:latest
docker build .\a-mmc_frontend\ -t krispybata/a-mmc-frontend:latest

2. cd .\a-mmc_infra\ ; docker compose up -d a-mmc-backend a-mmc-postgres a-mmc-frontend

3. In docker desktop, if both containers show green go to exec for a-mmc-postgres

4. psql -U postgres -d ammc_dev
