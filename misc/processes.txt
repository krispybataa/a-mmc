Processes: 
1. Doctor/Secretary 
	- Login/Logout 
	- Upload Doctor's Profile picture 
	- Manage Schedule (Weekly) 
	- View Appointments (and Appointment details) 
	- Accept Appointments 
		- Sends patients preliminiary information (like what to bring, how early to be, etc.) 
	- Request Reschedule of Appointments
		- Sends patients reason for reschedule 
		
|> Note on role: they need to be distinct from a org perspective, but technically have the same purpose and rights. 

2. Patient 
	- Login/Logout 
	- View dashboard 
	- View/Manage Appointments 
	- Update self (user) profile 
	- "Find a Doctor" 
		- Patient-aware: browse Doctor directory 
		- Patient-unaware: guided walkthrough (tl;dr mini-akinator) 
		- View doctor's page (Doctor's Profile) 
	- Book appoinment 
	- Request reschedule
